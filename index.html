My First Post

如何用 hugo 搭建个人博客 准备 拥有命令行程序 配置了git为环境变量 拥有github账号 一种markdown编辑器 步骤(可见hugo官方quick start) 去hugo官方下载适合本机版本的安装包，解压并添加到环境变量 使用命令行运行hugo version查看环境变量是否添加成功 创建一个文件夹，在命令行里输入hugo new site username.github.io-creator，即新建了hugo站点 cd至username.github.io-creator文件夹,输入git init新建本地仓库 执行git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 执行echo ‘theme = “ananke”’ » config.toml 执行hugo new posts/my-first-post.md 使用文本编辑器打开my-first-post.md 在原始代码下方输入博客内容，完成博客内容后将draft后的true改成false 执行hugo server -D 可在config.toml中修改配置 新建一个终端 在username.github.io-creator文件夹运行hugo 新建 .gitignore文件，在其中添加public目录 cd至public文件夹，依次执行git init ； git add . ； git commit -m “the first version” 登录github，新建仓库username.github.io 将public文件夹的本地分支push到github仓库 在github仓库点开settings，移动至Github Pages 点击个人博客链接则可查看博客内容
